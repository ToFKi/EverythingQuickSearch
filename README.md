## 📋 What is EverythingQuickSearch?

EverythingQuickSearch is a Windows app that helps you find files and folders quickly. It uses a simple search tool to replace your usual Windows search. This app is useful when you want to locate files without waiting for slow system searches.

It works by using the Everything search engine, which indexes your files instantly. Then you can type a few letters and see results immediately. The tool is light, fast, and built for everyday users.

## ❓ Troubleshooting

**The app won’t start:**  
Make sure Everything64.dll is placed next to the app executable. Both files must be in the same folder.

**Search results are empty:**  
Confirm that the Everything search service (version 1.4.1 or later) is running on your system. This service indexes your files. If you do not have it installed, you can download it from the Everything official site.

**App runs slow or freezes:**  
Restart your computer and reopen the app. Ensure your disk is not full or heavily used by other programs.
